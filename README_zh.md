## 📃介绍

一个基于终端的数学小工具。

本程序是免费开源的，此源代码可以随意使用。允许任何人使用源代码进行扩充，编译等操作。出现任何问题，本人不承担任何责任！

⚠注意：本程序处于开发阶段，不可用于科学的精密计算！如果出现任何问题，后果自负！

## ❔如何使用

在GitAction中寻找符合自己设备的版本

或自行拉取编译

📦编译环境需求：cmake, make，gcc
